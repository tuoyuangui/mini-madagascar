Program sfslice | Extract a slice using picked surface (usually from a stack or a semblance).
[SYNOPSIS]
sfslice < in.rsf pick=pick.rsf > out.rsf
[COMMENTS]

See also: sfpick.

June 2019 program of the month:
http://ahay.org/blog/2019/06/12/program-of-the-month-sfslice/

[PARAMETERS]
file    | pick | = |   | 	auxiliary input file name
